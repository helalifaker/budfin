# Documentation Archive

This directory stores historical working documentation that should remain available for traceability
but no longer needs to live in the active documentation set.

## Rules

- Keep the original filename when archiving.
- Group archived files by category, then year: `docs/archive/<category>/<year>/`.
- Do not archive authoritative sets such as `docs/prd/`, `docs/tdd/`, `docs/adr/`, `docs/specs/`,
  or `docs/ui-ux-spec/` unless a deliberate migration or replacement has been approved.
- Prefer moving files here over deleting them.
